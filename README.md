# FinalTerm Project

## Overview
YOLOv5 기반 쓰레기 객체 인식 시스템으로,
- Edge: YOLOv5 객체 인식
- Service: Django REST API 서버
- Client: Android 앱으로 결과 시각화

## Structure
- Edge_System/: YOLOv5 기반 객체 인식
- waste_server/: Django REST API 서버
- waste_client/: Android Studio 프로젝트
- FinalTerm.zip: 최종 PPT
- url.txt: 깃허브 url, pythonanywhere url

## Demo
- YOLOv5 → 서버 데이터 등록
- Android 앱에서 이미지 및 분류 결과 확인
